# OWASP API Security Top 10 — Coverage Checklist

Per-category status through Sprint 0. Sprints 1-5 extend this as new surface ships.

Reference: https://owasp.org/API-Security/editions/2023/en/0x11-t10/

| # | Category | Sprint 0 status | Notes |
|---|---|---|---|
| API1 | Broken Object-Level Authorization (BOLA) | ✅ addressed | Firestore rules (`infra/firestore.rules`) filter every read/write on `company_id`. `MemoryProvider` requires `company_id` positionally (Risk 12). |
| API2 | Broken Authentication | ✅ addressed | Firebase Admin SDK `verify_id_token` on every non-public request. 6 unit tests cover expired / tampered / missing / missing-company_claim / generic-ValueError paths. **Gap**: `check_revoked=True` not yet passed — revoked tokens pass until expiry. Sprint 4 hardening. |
| API3 | Broken Object Property Level Authorization | ✅ addressed | Pydantic `extra="forbid"` on every schema rejects unknown fields on input. Output schemas have `extra="forbid"` too — response envelopes can't leak extra properties by accident. |
| API4 | Unrestricted Resource Consumption | ⏸️ deferred to Sprint 4 | `RateLimitMiddleware` is a pass-through stub. Real enforcement (slowapi + Memorystore backend) lands Sprint 4 with `/triage/stream`. Cloud Armor layered on top at deploy (Sprint 5). |
| API5 | Broken Function-Level Authorization (BFLA) | ⏸️ deferred to Sprint 4 | No privileged routes yet. When admin routes land, use `require_tier(2)` dependency factory per `.claude/rules/security.md` §3. |
| API6 | Unrestricted Access to Sensitive Business Flows | ⏸️ deferred to Sprint 4 | No business flows exposed yet. `/triage/stream` rate limits + per-agent cost caps (observability §6) land together. |
| API7 | Server-Side Request Forgery (SSRF) | ✅ addressed | No outbound requests to user-controlled URLs in Sprint 0. When Sprint 2 Impact Agent adds external lookups (shipment portals), any URL comes from our Firestore-seeded data, not from user input. |
| API8 | Security Misconfiguration | ✅ addressed | CORS allowlist (wildcard rejected at startup); least-privilege IAM (dev SA has only `secretmanager.secretAccessor` + `datastore.user`); Secret Manager runtime fetch (not env baked into image); `uv.lock` committed + `uv sync --locked` in CI. |
| API9 | Improper Inventory Management | ⏸️ deferred to Sprint 4 | OpenAPI spec is auto-generated by FastAPI. Will be reviewed + versioned at Sprint 4. Tier 3 adds formal versioning (`/v1/...`). |
| API10 | Unsafe Consumption of APIs | ✅ addressed (Sprint 0 surface) | Every external response (Gemini, Supermemory, Firestore) parses through Pydantic schemas before entering business logic. No raw dicts cross the boundary — per `.claude/rules/api-routes.md` + `.claude/rules/tools.md` §1. |
